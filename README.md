# yumcha

yumcha is a macOS tiling window manager.

It is built to solve a couple of problems I ran into with existing tools:

- `yabai` is powerful, but it often pushes users toward disabling SIP.
- `aerospace` avoids some of that, but integration with existing window-management workflows feels disconnected from the rest of the system.

`yumcha` takes a different route: it talks directly to the macOS Accessibility APIs and tries to fit more naturally into the desktop experience.

The name comes from "yum cha": I took a little bit from a few different ideas and put them together into one project.
